How to setup a new Infrastructure for an E-commerce in
Traditional manner:
1. Planning for the needs of HW/SW.
Main Roles: Infrastructure Architect, Application architect, CIO/CTO

2. Identifying Correct specifications of HW and SW.
Main Roles: Technical lead/Managers, Infrastructure Architect, Application architect, CIO/CTO

3. You are suppose call different vendors to get the
quotes for HW/SW/Certificates for a defined architecture.
Main Roles: Technical lead/Managers, Vendor related people/SMEs, 
Infrastructure Architect, Application architect, CIO/CTO

4. Select the right vendors in view of cost, size and
support.
Main Roles: Technical lead/Managers, Infrastructure Architect, Application architect, CIO/CTO,
Financial Managers/CFO.

5. Issuing order.
Main Roles: CIO/CTO, Financial Managers/CFO.

6. Delivery/receiving the Items
Main Roles: Facilities Management Group, 
Technical lead/Managers, Infrastructure Architect, Application architect, 
CIO/CTO.

7. Setting up and installation of the HW/SW.

Main Roles: Technical lead/Managers, Vendor related people/SMEs, 
Infrastructure Architect, Application architect, CIO/CTO


8. Connecting them to Internet/Intranet.
Main Roles: Technical lead/Managers, Vendor related people/SMEs, 
Infrastructure Architect, Application architect, CIO/CTO


9. Operating for regular business.
Main Roles: Technical lead/Managers, Vendor related people/SMEs, 
Application architect, Ops managers/Leads, Business Users


10. Maintaining the HW/SW an doperational issues.
Main Roles: Technical lead/Managers, Vendor related people/SMEs
